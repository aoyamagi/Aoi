# To-Do 整理アプリ

スマホ最適化、PWA対応のシンプルなTo-Doアプリ。

## 特長

- 📱 モバイルファースト
- ✅ チェックで完了管理、アーカイブで非表示化
- 🎯 タスクごとに目的・🎒持ち物を整理
- 🏷️ カテゴリ＆優先度で整理
- 👥 JSONファイル / 共有URL / コピー&ペーストで複数人共有
- 💾 ブラウザに自動保存（オフライン動作）
- 📲 PWA対応：スマホのホーム画面に追加するとアプリのように起動

## 使い方

1. GitHub Pages 等で公開した URL をスマホで開く
2. ブラウザのメニューから「ホーム画面に追加」
3. アイコンをタップして起動

詳しい公開手順は [SETUP.md](./SETUP.md) を参照してください。

## ファイル構成

- `index.html` - アプリ本体
- `manifest.json` - PWA設定
- `sw.js` - Service Worker（オフライン用キャッシュ）
- `icon-192.png` / `icon-512.png` - アプリアイコン
- `make_icons.ps1` - アイコン生成スクリプト（任意）
